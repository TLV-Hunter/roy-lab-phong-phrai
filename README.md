# รอยลับแห่งพงไพร

เว็บไซต์คู่กับเพจ Facebook “รอยลับแห่งพงไพร” สำหรับรวบรวมเรื่องเล่าป่าพิศวง ตำนานไทย และสิ่งมีชีวิตมหัศจรรย์

## โครงสร้าง

- `index.html` — หน้าเว็บไซต์หลัก
- `styles.css` — รูปแบบและ Responsive Design
- `script.js` — เมนูสำหรับหน้าจอมือถือ

เว็บไซต์นี้ออกแบบให้เผยแพร่ผ่าน GitHub Pages และอัปเดตเรื่องใหม่ได้ในอนาคต
